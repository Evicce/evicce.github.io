Windows下安装Python环境

发表于 2018-09-22

1.安装Python解释器

访问Python官网下载Python安装包

打开安装程序，选中”Add Python 3.7 to PATH”选项，然后点击”Install Now”安装

安装成功

2.安装Pycharm IDE

访问Pycharm官网下载Pycharm安装包

打开安装程序，选择好安装路径，点击”Next”

根据自己的操作系统选择添加64位或32位的快捷方式到桌面，这里我的系统是64位所以选择”64-bit launcher”

一路Next后安装成功

3.第一次打开Pycharm

打开Pycharm，选中”Do not import settings”后，点击”OK”

根据你的喜好选择主题

4.创建一个项目

点击”Create New Project”来创建一个新项目

这里”Location”为项目保存路径，”Interpreter”为Python解释器路径，需要修改为你的Python安装路径

选中左侧的”System Interpreter”，修改右侧的路径为Python安装目录下的python.exe

点击”OK”后再点击”create”创建项目，出现一个Tip，点击”Close”关闭

右键项目文件夹创建一个Python文件

给文件起个名字

在右侧编辑区输入

快捷键ctrl+shift+F10来运行代码
